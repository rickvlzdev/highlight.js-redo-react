# Highlight.js Website Redo with React.js

## Installation
```
$ git clone https://github.com/rickvlzdev/highlight.js-redo-react.git hl-redo-react
$ cd hl-redo-react
$ npm install
```
## Start Application
```
$ npm run start
```
Open your preferred browser and go to http://localhost:8080. \
